The DOM

  • The DOM, or document object model, is a hierarchical way of laying out information. If, for example, we have a webpage in HTML like this:

    <!DOCTYPE html>
		<head>
			<title id="title">Hey Rob!</title>
		</head>
		<body>
			<div>
				<img id="pic" src="rob.jpg" alt="rob">
			</div>
			<div>
				<p id="quote">There is no happiness in the world, only rice...</p>
			</div>
		</body>
	</html>

    then its DOM will be a tree like this:

  • What’s cool about JS is we can change the DOM dynamically without having to reload the page! With code like this:

    <script>
	    var title = document.getElementById("title");
		title.innerHTML = "David";

		var pic = document.getElementById("pic");
		pic.src = "david.jpg";

		var alt = document.getElementById("pic");
		alt.alt = "david";

		document.getElementById("quote").innerHTML = "alllllright";
	</script>
